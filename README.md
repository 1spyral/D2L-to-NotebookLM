# D2L to NotebookLM

> Upload all your course materials to NotebookLM with one click.

**D2L to NotebookLM** is a browser extension that downloads your course content from D2L and uploads it directly into NotebookLM, transforming your classes into an AI-powered knowledge base.

Instead of digging through course pages, manually downloading files, and organizing folders, this extension automates the process so you can focus on understanding the material.

---

## Features

- **Extract** course materials from D2L
- **Upload** directly into NotebookLM
- **One-click** workflow

---

## Who Is This For?

- Students managing multiple courses
- Exam prep & quick review
- Building AI-assisted study guides
- Anyone who wants a smarter way to study

---

## Privacy & Security

Your data belongs to you.

- Files are processed locally in your browser
- Only selected content is uploaded
- No tracking, analytics, or hidden data collection
- NotebookLM authentication is handled securely

> [!WARNING]
> Always ensure you comply with your school's academic integrity and content policies.

## Installation (Development)

**Setup**

```bash
pnpm install
```

**Build**

```bash
pnpm run build
```

Build outputs:

| Target  | Output path    |
|---------|----------------|
| Chrome  | `dist/chrome`  |
| Firefox | `dist/firefox` |
| Safari  | `dist/safari`  |
